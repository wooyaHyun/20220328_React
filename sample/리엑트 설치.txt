
1. 확인
    npm info create-react-app

2. 설치
    리엑트 프로젝트를 생성하는 보일러 플레이트 설치
    어느 경로에서든지 프로젝트를 생성할 수 있도록 글로벌에 설치한다

    npm i -g create-react-app

    설치 버전 확인
        create-react-app --version

3. 프로젝트 생성    
    create-react-app 프로젝트이름

    4.0.3이 설치되어 있는 경우 프로젝트 생성에 실패하면 아래와 같은 방법으로 생성. 5버전으로 설치된다
    create-react-app@latest project-name

    프로젝트를 타입 스크립트 기반으로 생성하는 경우.
    npx create-react-app@latest project-name --template typescript
    
4. 프로젝트로 진입
    cd 프로젝트이름
    
5. 프로젝트 시작
    npm start

6. 프로젝트 종료
    ctrl + c

7. 수업 예제는 bootstrap으로 작성되어 있어 bootstrap을 기본 template403에 먼저 설치
    npm i bootstrap@4       => 4의 마지막 버전
    npm i bootstrap@4.5     => 4.5의 마지막 버전
    npm i bootstrap@4.5.1   => 매칭되는 버전 설치



파일 정렬
    Alt + Shift + F
